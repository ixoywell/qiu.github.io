Markdown语法说明

发表于 2016-05-19   |   分类于 目录   |  

Markdown 是一种用来写作的轻量级「标记语言」，它用简洁的语法代替排版，而不像一般我们用的字处理软件 Word 或 Pages 有大量的排版、字体设置。它使我们专心于码字，用「标记」语法，来代替常见的排版格式。

Markdown语法说明

使用 Markdown 的优点

1.专注你的文字内容而不是排版样式。
2.轻松的导出 HTML、PDF 和本身的 .md 文件。
3.纯文本内容，兼容所有的文本编辑器与字处理软件。
4.可读，直观。适合所有人的写作语言。
